网易娱乐7月20日报道 7月15日，耗资7.5亿、上映仅三天的电影《阿修罗》宣布撤档。电影上映前，《阿修罗》制片人杨真鉴接受采访说，投资7.5亿不仅不多，而且喊话谁能再给2000万美金，他不仅能颠覆中国电影，而且至少能跟好莱坞较牛电影持平。

采访中，杨真鉴被问到电影是否符合自己的预期，他表示：“首先我提出的一个目标，就是（它能成为）引领中国产业半步的一部电影，初步达到好莱坞A级片入门的电影。”他还透露，好莱坞A级电影的入门电影，投资是1亿美金，且没有任何超一流大咖和超复杂视效；而投资到了2亿到2.5亿时，就能做出《魔兽》的水准；到了3、4亿，就是《哈利波特》或者《阿凡达》的水准，这是一个产业，不是想省就省。

他还表示，其他制片人“你们没花过这个钱，你们太没有见识了”，更喊话投资人：“这部电影（投资）7.5亿不仅不多，谁能再给我2000万美金，我真能把他们（好莱坞）颠覆了：不是颠覆中国电影，是至少跟好莱坞较牛的电影持平这个水准。”
